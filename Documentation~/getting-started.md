# Getting Started

## Prerequisites

- Unity 6000.0 (Unity 6.0 LTS) or later
- A Figma account with a Personal Access Token (PAT)
- The Figma UI Toolkit Generator plugin installed in your Figma workspace

## Installation

### Step 1 -- Add via Git URL

Open your Unity project's `Packages/manifest.json` and add:

```json
{
  "dependencies": {
    "com.bizsim.unity.figma.importer": "https://github.com/BizSim-Game-Studios/com.bizsim.unity.figma.importer.git#v1.0.1"
  }
}
```

Alternatively, use **Window > Package Manager > + > Add package from git URL** and paste:

```
https://github.com/BizSim-Game-Studios/com.bizsim.unity.figma.importer.git#v1.0.1
```

### Step 2 -- Configure your Figma token

1. Go to **Edit > Preferences** (macOS: **Unity > Settings**).
2. Find the **Figma Importer** section.
3. Paste your Figma Personal Access Token.

To generate a PAT: open Figma, go to **Account Settings > Personal Access Tokens**, and create a new token with read access.

### Step 3 -- Create an Importer Settings asset

1. Right-click in the Project window.
2. Select **Create > BizSim > Figma Importer Settings**.
3. Configure path rules for your asset categories (see [Configuration](configuration.md)).

## First Import

1. In your Figma file, run the Figma UI Toolkit Generator plugin to produce UXML/USS and the Asset Manifest JSON.
2. Copy the Asset Manifest JSON into your Unity project (e.g., `Assets/FigmaManifest.json`).
3. Open the Figma Importer window (if available) or use the API:

```csharp
using BizSim.Unity.Figma.Importer.Editor;

var settings = AssetDatabase.LoadAssetAtPath<FigmaImporterSettings>(
    "Assets/Settings/FigmaImporterSettings.asset");
var orchestrator = new FigmaImportOrchestrator(settings);

string json = File.ReadAllText("Assets/FigmaManifest.json");
var manifest = orchestrator.ParseManifest(json);
var items = orchestrator.ScanProject(manifest);

var result = await orchestrator.Import(items, "YOUR_FIGMA_FILE_KEY");
Debug.Log($"Imported {result.imported}/{result.totalAssets} assets");
```

4. After import, the UXML/USS files generated by the plugin should render correctly with the downloaded images and fonts.
